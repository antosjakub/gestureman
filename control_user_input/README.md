

# wayland-automation

https://pypi.org/project/wayland-automation/


## wayland-utils, wtype, wl-find-cursor


```
sudo dnf install wayland-utils
sudo dnf install wtype
```

build the other one:

```
git clone https://github.com/cjacker/wl-find-cursor.git
cd wl-find-cursor
make
sudo cp wl-find-cursor /usr/local/bin/
```
